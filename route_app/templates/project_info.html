Welcome to the Route Optimization Project

Optimize your travel routes efficiently using advanced algorithms.

About This Project

This application is designed to help users find the most efficient travel route between a set of cities. By leveraging different optimization algorithms, this system enables users to calculate and visualize optimal routes based on a dataset of city distances.

Project Features

Algorithm Selection

Choose from multiple algorithms to calculate the best route, including Ant Colony Optimization, Floyd-Warshall, and more.

Interactive Inputs

Select cities from dropdowns that automatically update based on your previous choices, ensuring no duplicate entries.

Detailed Visualization

Visualize the optimal route on a graph, along with statistics like total distance and time taken for each algorithm.

Technologies Used

  • Python and Django - Backend and Framework
  • Pandas and NumPy - Data Manipulation and Calculations
  • Matplotlib - Analyzing Execution time in milliseconds
  • Bootstrap - UI Framework for Responsive Design
